Blitzprog SDK - Changelog
-------------------------

Version 0.0.3.0
 -> XML-Parser hinzugefügt
 -> Grafik-Modul hinzugefügt
 -> LoadImage zum Laden von Bildern (zurzeit nur aus vorher geladenen Pixmaps)
 -> DrawImage zum Zeichnen von Bildern
 -> DrawRect, DrawLine und DrawPixel fertiggestellt
 -> SetRotation um die Rotation zu beeinflussen
 -> SetScale verändert den Skalierungsfaktor (individuell für lokale x- und y-Achse)
 -> SetColor setzt die aktuelle Zeichenfarbe
 -> Vector2D-Klasse (Synonym: Point) erstellt
 -> InitGraphics hinzugefügt, kann von BB-Umsteigern genutzt werden
 -> MouseX() und MouseY() hinzugefügt, ermitteln die zum aktuellen Fenster relative Mausposition (nur im Grafikmodus)
 -> KeyDown und KeyHit hinzugefügt
 -> Die Klassen Input und InputAxis hinzugefügt (bieten mehr Möglichkeiten als KeyDown)
 -> List-Klasse hinzugefügt (im Modul "Collection")
 -> 4 neue Funktionen im Mathematik-Modul: IsPowerOfTwo, IsPythagoreanTriple, Factorial und BinomialCoefficient
 -> UpdateFPS ist nun thread-safe und funktioniert korrekt
 -> Directory-Klasse und LoadDir-Funktion zum Auslesen aller Dateien in einem Verzeichnis hinzugefügt
 -> Color-Klasse besitzt nun Get- und Set-Funktionen für alle Farbkomponenten
 -> Die Klasse Array ist nun vom Collection-Interface abgeleitet und bietet deutlich mehr Funktionen
 -> Kompilierzeit verringert ("große" Funktionen, die sowieso nicht "geinlined" werden können, wurden aus den Header-Dateien entfernt)
 -> Die Funktion End() löscht nun alle vorher erzeugten Variablen und akzeptiert auch einen Exit-Code als Parameter
 -> Bugfix: Alle Get-Funktionen von TWindow werden nun synchronisiert
 -> Bugfix: TArray::ToString entfernt, weil nicht jede Klasse in einen String konvertiert werden kann
 -> Bugfix: Alpha-Blending funktioniert nun korrekt

Version 0.0.2.1
 -> GUI-Modul hinzugefügt (zur Zeit noch im Anfangsstadium)
 -> Kollisions-Modul im Bereich Mathematik hinzugefügt, sowohl 2D als auch 3D (ebenfalls noch im Anfangsstadium)
 -> Thread-Library von pthread auf boost::thread umgestellt (komplett fertig)
 -> Vector3D-Klasse hinzugefügt
 -> Modul für Kollisionsfunktionen (2D) im Bereich Mathematik hinzugefügt
 -> CurrentDate() und CurrentTime() im Datumsmodul implementiert

Version 0.0.2.0
 -> Komplett neu geschrieben
 -> Alle Module optimiert und ToDo's erledigt
 -> Grafik- und GUI-Modul temporär entfernt, weil ich mich erstmal um einen stabilen Kern kümmern muss
 -> String-Klasse so gut wie fertig
 -> Netzwerk-Modul beinhaltet eine TCPStream-Klasse (zur Zeit nur für Clients geeignet, Server wird später implementiert)
 -> Einfach zu nutzende Thread-Klasse für Multithreading
 -> Abstrakte Stream-Klasse fast fertig
 -> Dateisystemfunktionen und FileWriteStream/FileReadStream-Klassen hinzugefügt
 -> Zur Zeit nur unter Linux lauffähig, bei Windows müssen noch einige Dinge geändert werden
